# Pyramid Developer

Conventions-only skill that installs Pyramid framework conventions as a glob-scoped rule (active on `**/*.py`). Covers the configuration/registry system, `includeme` module pattern, `pyramid_services`, SQLAlchemy models, Cornice REST, ACL authorization, and transaction management.

This is the shared foundation for all three Pyramid developer personas:

- **App Developer** — building and running a Pyramid application (composition root, test infrastructure, pshell snippets). See the **pyramid-app** skill.
- **Plugin Developer** — writing domain modules inside the app (module anatomy, `includeme`, service layer, REST endpoint scaffolding). See the **pyramid-plugin** skill.
- **Library Developer** — developing standalone Pyramid extensions in their own repo (custom directives, tweens, settings, consumer-facing `includeme`). See the **pyramid-library** skill.
