# Pyramid Library (Library Developer)

Skill for the **Library Developer** role — developing standalone Pyramid extensions that live in their own repository and are distributed as independent Python packages. These libraries expose `includeme` for consumers to `config.include()`, and must not assume anything about the host application's internals.

If you're in the **App Developer** role (composition root, test infrastructure, pshell snippets), see **pyramid-app**. If you're in the **Plugin Developer** role (domain modules inside the app codebase, services, REST endpoints), see **pyramid-plugin**. For shared Pyramid conventions (registry, models, authorization, transactions), see the **pyramid-developer** always-on rule.
